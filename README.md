# runnerspace

A retro myspace clone for UTSA students.

## About

Made during Rowdy Hacks 2022, this myspace look-a-like was quickly made with intentions of serving
the UTSA student community.

## Tech Stack

- Flask
  - Jinja Templating
  - SQLAlchemy + SQLite
- Sass
- Font Awesome Icons
- Other
  - Humanize
  - Faker

# Setup

Use Python 3.8.

```bash
pip install pipenv
pipenv install
flask run
```
